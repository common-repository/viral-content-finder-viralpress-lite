ViralPress Lite is the most powerful and unique WordPress plugin to find and analyze viral content from Facebook for any business/niche.

== Description ==

ViralPress Lite is a very easy to use WordPress plugin to find Viral content from Facebook. ViralPress Lite helps you in finding and analyzing the content which went Viral in your business niche. Within a few seconds, you can easily find why is a particular Facebook page so popular by exploring its most popular (Viral) content using ViralPress Lite plugin.

Step by step documentation is here : https://indigothemes.com/documentation/viralpress-lite/

ViralPress Pro Features can be seen here : https://indigothemes.com/products/viralpress-pro-viral-content-finder-wordpress-plugin/
 


== Installation ==

This section describes how to install the ViralPress Lite plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/viral-content-finder-viralpress-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->ViralPress Lite screen to configure the Facebook App settings.


== Frequently Asked Questions ==

= What is the documentation URL ? =

Documentation URL is https://indigothemes.com/documentation/viralpress-lite/

= What is the main difference between ViralPress Lite and ViralPress Pro plugin ? =

ViralPress Pro is the advanced version of ViralPress Lite. ViralPress Lite has limitations that you only get 10 images/videos when you explore a particular page but in ViralPress Pro, you get unlimited images/videos. ViralPress Pro has filters using which you can sort the data based on total likes and shares and much more. 


= Why should I try ViralPress Lite ? =

If you're a small or medium sized business owner then you definitely want your Facebook page, Twitter account to have loads of fans and followers. Using ViralPress Lite, you can find what type of content is going Viral on Facebook related to your business niche. You can then create similar type of content and share it on social media platforms to get more fans/followers.

= How can you so sure that ViralPress Lite work ?  =

It's based on a simple principal that - what went Viral in the Past will definitely go Viral in the future too. Human behaviour is very much predictable in this regard. We all love something Unique and Special and ViralPress Lite helps us in finding such type of content.
